# 🌐 3D Portfolio – Built with Three.js & React

Welcome to my interactive 3D portfolio built using **Three.js** and **React**! This site is a dynamic showcase of my projects and creative work, blending real-time 3D graphics with modern frontend technologies.

## ⚙️ Tech Stack

- **Node.js** – Backend and development tooling
- **React.js** – Frontend framework
- **Three.js** – WebGL-based 3D engine
- **React Three Fiber (R3F)** – React renderer for Three.js
- **React Three Drei** – Helpers and abstractions for R3F
- **EmailJS** – Contact form integration
- **Vite** – Fast development and build tool
- **Tailwind CSS** – Utility-first styling

## 🚀 Features

- 🧭 Smooth 3D scene navigation and camera controls
- ✨ Custom shaders and lighting
- 📱 Fully responsive UI with Tailwind CSS
- 📨 Contact form powered by EmailJS
- 📦 Optimized asset loading (GLTF models, textures)
- ⚡️ Fast builds and hot reload via Vite
- 🎮 Interactive elements and animations using Drei + R3F

🧑‍💻 Getting Started
Prerequisites

Node.js v16+
npm or yarn

✍️ Customization

You can easily update models, lighting, and styles:
3D models: src/assets/
Scenes & animation: src/scenes/
UI components: src/components/
Styling: Tailwind via className or src/index.css
