## Overview

This is a 3D interactive portfolio site built with `React`, `Three.js`, and `TailwindCSS`.
This project uses `Vite` to achieve a high-speed development environment.

## Installation and Usage

### 1. Clone the repository

```bash
git clone https://github.com/vanshsuri07/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Access `http://localhost:5173/` to see the application.
